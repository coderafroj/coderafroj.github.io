HTML Basics

1. What is HTML?

HTML (HyperText Markup Language) is the standard language for creating web pages. It structures web content using elements like headings, paragraphs, links, images, and more.

2. Basic Structure of an HTML Page

Every HTML document starts with a basic structure:

<!DOCTYPE html>
<html>
    <head>
        <title>My First Web Page</title>
    </head>
    <body>
        <h1>Welcome to HTML!</h1>
        <p>This is a paragraph.</p>
    </body>
</html>

3. HTML Elements

HTML elements are the building blocks of a webpage. Example:

  • <h1>...</h1> - Headings
  • <p>...</p> - Paragraph
  • <a href="URL">...</a> - Links
  • <img src="image.jpg" alt="description"> - Images

4. Headings & Paragraphs

HTML provides six levels of headings:

<h1>Heading 1</h1>
<h2>Heading 2</h2>
<h3>Heading 3</h3>
<p>This is a paragraph.</p>

5. Lists in HTML

There are two main types of lists:

Ordered List:

<ol>
    <li>Item 1</li>
    <li>Item 2</li>
</ol>

Unordered List:

<ul>
    <li>Item 1</li>
    <li>Item 2</li>
</ul>

6. Links and Anchors

HTML links are created using the <a> tag.

<a href="https://www.bytecorecomputercentre.com">Visit Bytecore</a>

7. Images in HTML

Use the <img> tag to add images.

<img src="image.jpg" alt="Description">

8. Tables in HTML

<table border="1">
    <tr>
        <th>Name</th>
        <th>Age</th>
    </tr>
    <tr>
        <td>John</td>
        <td>25</td>
    </tr>
</table>

9. Forms in HTML

<form>
    Name: <input type="text" name="name">
    <input type="submit" value="Submit">
</form>

3. HTML APIs (Drag & Drop, Notifications, Speech API)

✔️ Drag & Drop Feature

<div id="dragMe" draggable="true" ondragstart="drag(event)">Drag me</div>
<div id="dropHere" ondrop="drop(event)" ondragover="allowDrop(event)" style="border:1px solid; width:200px; height:50px;">Drop Here</div>

<script>
function allowDrop(event) {
    event.preventDefault();
}

function drag(event) {
    event.dataTransfer.setData("text", event.target.id);
}

function drop(event) {
    event.preventDefault();
    var data = event.dataTransfer.getData("text");
    event.target.appendChild(document.getElementById(data));
}
</script>

✔️ Web Notifications

</script>

</script>

HTML Attributes -

🔹 HTML Attributes - Detailed Overview

HTML Attributes, elements ke additional properties define karte hain jo unhe specific behavior ya styling provide karte hain. Yeh attributes element tags ke andar likhe jaate hain aur unke values ko specify karte hain.

📌 1. Basic Structure of HTML Attribute

<img src="image.jpg" alt="Image description" width="500" height="300">

Yahan `src`, `alt`, `width`, aur `height` attributes hai jo `` element ke properties define karte hain.

📌 2. Common HTML Attributes

  • id - Element ka unique identifier.
  • class - CSS styling apply karne ke liye use hota hai.
  • src - External resource ka path define karta hai, jaise images ya scripts.
  • href - Hyperlink ke liye destination URL specify karta hai.
  • alt - Image ke liye alternative text, jo accessibility ke liye important hai.
  • title - Element ka additional information display karta hai jab user hover kare.

📌 3. HTML Global Attributes

  • lang - Language specify karta hai (e.g., lang="en" for English).
  • style - Inline CSS styles define karta hai.
  • tabindex - Keyboard navigation order specify karta hai.
  • data-* - Custom attributes jo specific data store karne ke liye use hote hain.

📌 4. Important HTML Event Attributes

  • onclick - Element par click hone par JavaScript function ko trigger karta hai.
  • onmouseover - Mouse hover hone par function execute hota hai.
  • onchange - Form elements ke value change hone par function call hota hai.
  • onfocus - Element focus hone par event trigger hota hai (form input).

📌 5. Advanced HTML Attributes Usage

Example: data-* attributes

<div data-user="Afu" data-role="developer">User Information</div>

Yeh custom data attributes HTML tags ke andar additional information store karne ke liye use hote hain. JavaScript se hum in values ko access kar sakte hain:

document.querySelector("div").dataset.user; // "Afu"

📌 6. HTML Forms and Attributes

Forms me attributes form elements ko behavior aur validation control karte hain:

  • action - Form submission ka destination URL.
  • method - Form submission method (GET/POST).
  • required - Form field ko mandatory banata hai.
  • placeholder - Form input field me default text display karta hai.
<form action="submit.php" method="POST">
    <input type="text" required placeholder="Enter your name">
    <button type="submit">Submit</button>
</form>

📌 7. HTML5 New Attributes

  • autofocus - Input field ko automatically focus karta hai jab page load ho.
  • placeholder - Field ke andar placeholder text display karta hai.
  • autocomplete - Form input field ke liye browser ko auto-fill suggest karta hai.
  • required - Field ko mandatory banaata hai.

📌 8. HTML Accessibility Attributes

HTML me accessibility attributes use karna zaroori hai taaki visually impaired users ko bhi website use karna asaan ho:

  • aria-label - Element ko descriptive label deta hai, jo screen readers use karte hain.
  • role - Element ka role define karta hai, jaise role="navigation".
  • tabindex - Keyboard users ke liye tab navigation define karta hai.
<button aria-label="Close Menu">X</button>

🔹 Conclusion:

HTML attributes website ke structure aur behavior ko customize karte hain. In attributes ko samajhna zaroori hai, kyunki yeh **web development ke fundamental parts** hain. Agar aapko attributes achhe se samajh aa gaye, to aap easily **advanced HTML features** ka use kar sakte hain.

  • <table> - Table
